Aplikasi Arsip Digital

Homepage

Selamat datang di Aplikasi E-Arsip. Aplikasi ini digunakan untuk mengunggah dan mengelola dokumen arsip secara digital. Semua file yang diunggah akan disimpan di folder arsip resmi kantor.

Privacy Policy

Aplikasi Arsip hanya mengakses file yang Anda pilih untuk diunggah ke folder arsip kantor. Data yang diunggah tidak dibagikan kepada pihak ketiga dan hanya digunakan untuk kepentingan arsip internal Dinas Kantor.

Aplikasi menggunakan Google Drive API dan Google Sheets API untuk menyimpan dan mengelola file dan metadata arsip. Informasi login akun Google Anda hanya digunakan untuk autentikasi dan tidak disimpan oleh aplikasi.

Terms of Service

Pengguna bertanggung jawab memastikan file yang diunggah sesuai dengan peraturan dan kebijakan internal. Dinas Kantor tidak bertanggung jawab atas file yang diunggah secara tidak sah oleh pengguna.
